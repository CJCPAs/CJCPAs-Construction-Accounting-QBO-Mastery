**How to Customize:**

```
Click: Customize (upper right)

General:
• Report period: This Year (or custom dates)
• Accounting method: Accrual
• Number format: Divide by 1000 (optional for large numbers)

Rows/Columns:
• Columns: Classes
• Show non-zero or active rows only: ✓
• Show % of Row: ✓ (adds percentage columns)

Filter:
• Classes: Select specific jobs or "All"
• Active classes only: ✓ (hides completed jobs)

Header/Footer:
• Show logo: As desired
• Report title: Customize if needed

Click: Run Report

To save as favorite: Save customization → Name it → Add to favorites
```

**Key Construction-Specific Lines:**

```
WATCH THESE ACCOUNTS:

ASSETS:
• Retainage Receivable - Money customers owe you after releasing retention
• Costs in Excess (Underbilling) - You've spent more than billed (WIP asset)

LIABILITIES:
• Retainage Payable - Money you owe subs after releasing retention
• Billings in Excess (Overbilling) - You've billed more than earned (WIP liability)

The difference between over and underbilling shows if you're:
• Net underbilled: Financing customers' projects (need more working capital)
• Net overbilled: Customers financing your operations (better cash position)
```

## Work in Progress (WIP) Reports

### WIP Schedule (Manual/Spreadsheet)

QBO doesn't generate WIP schedules automatically. You'll need to compile data:

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                    WIP SCHEDULE                                              │
│                    As of March 31, 2024                                      │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│  Job        │Contract│ Est    │Costs   │% Comp│Earned  │Billed  │Over/Under│
│             │ Value  │ Cost   │to Date │      │Revenue │to Date │          │
│  ───────────┼────────┼────────┼────────┼──────┼────────┼────────┼──────────│
│  Smith Res  │$450,000│$350,000│$133,000│ 38.0%│$171,000│$180,000│ ($9,000) │
│  Johnson    │$280,000│$225,000│ $80,500│ 35.8%│$100,240│ $95,000│   $5,240 │
│  Martinez   │$325,000│$260,000│ $35,500│ 13.7%│ $44,525│ $45,000│    ($475)│
│  ───────────┼────────┼────────┼────────┼──────┼────────┼────────┼──────────│
│  TOTAL      │$1,055K │$835,000│$249,000│      │$315,765│$320,000│ ($4,235) │
│                                                                              │
│  Legend:                                                                     │
│  • Over = Overbilled (negative) = Billed more than earned = Liability       │
│  • Under = Underbilled (positive) = Earned more than billed = Asset        │
│                                                                              │
│  Net Position: ($4,235) Overbilled                                          │
│  This means company has billed $4,235 more than earned                      │
│                                                                              │
└─────────────────────────────────────────────────────────────────────────────┘

STEPS TO CREATE:
1. Pull P&L by Class for contract value, costs to date
2. Pull customer balance/billing history for billed to date
3. Calculate % complete: Costs to Date / Estimated Total Cost
4. Calculate Earned Revenue: Contract Value × % Complete
5. Calculate Over/Under: Billed - Earned
```

### Profit and Loss by Class for WIP Data

```

Compare to:

### Key Performance Indicators (KPIs)

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                    CONSTRUCTION KPIs                                         │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│  PROFITABILITY:                                                              │
│  ───────────────                                                             │
│  Gross Margin by Job = (Revenue - Direct Costs) / Revenue                   │
│  Target: 20-30% for most contractors                                        │
│                                                                              │
│  Net Profit Margin = Net Income / Revenue                                   │
│  Target: 5-10% for healthy contractor                                       │
│                                                                              │
│  LIQUIDITY:                                                                  │
│  ───────────                                                                 │
│  Current Ratio = Current Assets / Current Liabilities                       │
│  Target: 1.25 or higher                                                      │
│                                                                              │
│  Quick Ratio = (Cash + AR) / Current Liabilities                            │
│  Target: 1.0 or higher                                                       │
│                                                                              │
│  WORKING CAPITAL:                                                            │
│  ────────────────                                                            │
│  Working Capital = Current Assets - Current Liabilities                     │
│  Target: Positive and growing                                                │
│                                                                              │
│  EFFICIENCY:                                                                 │
│  ───────────                                                                 │
│  AR Days = (AR / Annual Revenue) × 365                                      │
│  Target: Under 45 days                                                       │
│                                                                              │
│  AP Days = (AP / Annual COGS) × 365                                         │
│  Target: Match payment terms (30-45 days)                                   │
│                                                                              │
│  BACKLOG:                                                                    │
│  ────────                                                                    │
│  Backlog = Contract Value - Billed to Date (for active jobs)                │
│  Backlog Months = Backlog / Average Monthly Revenue                         │
│  Target: 6-12 months of backlog                                              │
│                                                                              │
│  OVERBILLING/UNDERBILLING:                                                   │
│  ──────────────────────────                                                  │
│  Net Under/Over = Costs in Excess - Billings in Excess                      │
│  Target: Slight overbilling (positive cash flow)                            │
│                                                                              │
└─────────────────────────────────────────────────────────────────────────────┘
```

### Exporting and Sharing Reports

```
FROM QBO REPORT:

1. EMAIL
   Click: Email → Enter recipients → Send

   Click: Export → Export to Excel
   
3. EXPORT TO PDF
   Click: Export → Export to PDF (or Print → Save as PDF)

4. SCHEDULED REPORTS (QBO Advanced)
   Navigate: Reports → Custom Reports → Scheduled
   Set: Frequency, recipients, format
   
FOR BONDING/BANKING:
