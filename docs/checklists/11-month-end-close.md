# Month-End Close Checklist for Construction Companies

## Overview

A consistent month-end close process ensures your financial statements are accurate, timely, and useful for decision-making. This checklist provides a comprehensive, step-by-step process tailored for construction companies using QuickBooks Online.

---

## Close Calendar

### Recommended Timeline

```
MONTH-END CLOSE CALENDAR

Day 1-3:   Gather and enter outstanding data
Day 4-5:   Complete transaction entry
Day 6-7:   Run preliminary reports, identify issues
Day 8-9:   Make adjusting entries
Day 10:    Final review and "soft close"
Day 11+:   Close period, distribute reports
```

### Key Dates to Track

| Activity | Target Date | Responsible |
|----------|-------------|-------------|
| Timesheets due | Last day of month | Field staff |
| AP entry cutoff | 3rd business day | Bookkeeper |
| Bank rec complete | 5th business day | Bookkeeper |
| WIP meeting | 6th business day | PM's + Controller |
| Preliminary review | 8th business day | Controller |
| Books closed | 10th business day | Controller |
| Report distribution | 11th business day | Controller |

## Pre-Close Preparation (Day 1-3)

### Gather Outstanding Information

- [ ] **Timesheets**
  - Collect all timesheets for the month
  - Follow up on missing timesheets
  - Verify hours are coded to correct projects
  - Note any unbilled overtime

- [ ] **Vendor Invoices**
  - Request copies of any missing invoices
  - Review mail for late-arriving bills
  - Check email for electronic invoices
  - Review vendor statements for discrepancies

- [ ] **Subcontractor Pay Apps**
  - Collect all sub pay applications
  - Verify work completed matches billing
  - Note retention amounts
  - Confirm lien waivers attached

- [ ] **Project Documentation**
  - Gather approved change orders
  - Collect field work orders
  - Review pending change order status
  - Update contract values if COs approved

- [ ] **Bank Statements**
  - Download all bank statements
  - Obtain credit card statements
  - Get loan statements (monthly)
  - Access online banking for transactions

---

## Transaction Entry Completion (Day 4-5)

### Accounts Payable

- [ ] **Enter All Bills**
  - Enter all vendor invoices received
  - Code to correct expense accounts
  - Assign to correct projects
  - Record any retention payable
  - Apply early payment discounts if applicable

- [ ] **AP Review**
  - Review AP aging for duplicates
  - Verify large invoices are properly coded
  - Check for unusual vendors
  - Confirm intercompany items

### Accounts Receivable

- [ ] **Enter All Invoices/Progress Billings**
  - Prepare and enter progress billings
  - Enter any additional invoices
  - Record retention receivable
  - Apply customer payments received

- [ ] **AR Review**
  - Review AR aging for accuracy
  - Identify delinquent accounts
  - Note collection actions needed
  - Verify large payments applied correctly

### Payroll

- [ ] **Verify Payroll Entry**
  - Confirm all payrolls posted
  - Check labor distribution by project
  - Verify burden allocations
  - Reconcile to payroll reports

### Bank Transactions

- [ ] **Categorize All Transactions**
  - Download bank feed transactions
  - Categorize all entries
  - Assign to projects where applicable
  - Clear pending items

---

## Reconciliations (Day 6-7)

### Bank Reconciliations

- [ ] **Complete All Bank Recs**
  - Reconcile operating account
  - Reconcile payroll account
  - Reconcile savings/money market
  - Document any reconciling items

**Bank Reconciliation Checklist:**
```
Account: _________________ Month: _________

Statement Balance:         $__________
Add: Deposits in Transit   $__________
Less: Outstanding Checks   $__________
Adjusted Bank Balance:     $__________

QBO Balance:              $__________
Difference:               $__________  (Should be $0.00)

Reconciled By: ________ Date: ________
```

### Credit Card Reconciliations

- [ ] **Reconcile All Cards**
  - Match statement to QBO balance
  - Review all charges for proper coding
  - Identify any personal charges
  - Note unusual or duplicate charges

### Loan Reconciliations

- [ ] **Reconcile Loan Balances**
  - Verify loan balance matches statement
  - Confirm interest vs. principal split
  - Update amortization schedules
  - Record any new borrowing

---

## Project Review & WIP (Day 7-8)

### Cost-to-Complete Meeting

**Meeting Participants:** Project Managers, Controller, Estimating

- [ ] **For Each Active Project:**
  - Review costs incurred to date
  - Update estimated cost to complete
  - Discuss any problem areas
  - Identify pending change orders
  - Note any billing issues

**Cost-to-Complete Worksheet:**
```
Project: ________________________

Original Contract:           $__________
Approved Change Orders:      $__________
Current Contract Value:      $__________

Original Est. Cost:          $__________
Costs to Date:               $__________
Est. Cost to Complete:       $__________
Revised Total Est. Cost:     $__________

Est. Gross Profit:           $__________
Est. Gross Margin:           _________%

Notes/Issues:
_________________________________
_________________________________
```

### WIP Schedule Preparation

- [ ] **Complete WIP Schedule**
  - Enter all project data
  - Calculate % complete for each job
  - Calculate earned revenue
  - Calculate over/under billing
  - Identify jobs with concerns

- [ ] **WIP Analysis**
  - Review for reasonableness
  - Compare to prior month
  - Flag significant changes
  - Document explanations

### WIP Adjusting Entry

- [ ] **Calculate WIP Adjustment**
  ```
  Prior Month CIE Balance:     $__________
  Current Month CIE Required:  $__________
  CIE Adjustment Needed:       $__________

  Prior Month BIE Balance:     $__________
  Current Month BIE Required:  $__________
  BIE Adjustment Needed:       $__________
  ```

- [ ] **Record WIP Journal Entry**
  ```
  If CIE increased (more under-billing):
    Dr. Costs in Excess of Billings
    Cr. Contract Revenue

  If CIE decreased (less under-billing):
    Dr. Contract Revenue
    Cr. Costs in Excess of Billings

  If BIE increased (more over-billing):
    Dr. Contract Revenue
    Cr. Billings in Excess of Costs

  If BIE decreased (less over-billing):
    Dr. Billings in Excess of Costs
    Cr. Contract Revenue
  ```

## Adjusting Entries (Day 8-9)

### Standard Monthly Entries

- [ ] **Depreciation**
  - Calculate monthly depreciation
  - Record depreciation entry
  - Update depreciation schedules

  ```
  Dr. Depreciation Expense - Vehicles     $X,XXX
  Dr. Depreciation Expense - Equipment    $X,XXX
  Dr. Depreciation Expense - Building     $X,XXX
    Cr. Accumulated Depreciation - Vehicles      $X,XXX
    Cr. Accumulated Depreciation - Equipment     $X,XXX
    Cr. Accumulated Depreciation - Building      $X,XXX
  ```

- [ ] **Prepaid Expenses**
  - Amortize insurance premiums
  - Expense prepaid rent if applicable
  - Amortize any other prepaids

  ```
  Dr. Insurance Expense                   $X,XXX
    Cr. Prepaid Insurance                        $X,XXX
  ```

- [ ] **Accrued Expenses**
  - Accrue unpaid wages (if applicable)
  - Accrue vacation earned
  - Accrue interest on loans
  - Accrue any other known expenses

  ```
  Dr. Wages Expense                       $X,XXX
    Cr. Accrued Wages                           $X,XXX

  Dr. Interest Expense                    $X,XXX
    Cr. Accrued Interest                        $X,XXX
  ```

- [ ] **Equipment Allocation**
  - Review equipment usage logs
  - Calculate job allocations
  - Record equipment allocation entry

### Project-Specific Entries

- [ ] **Retention Adjustments**
  - Verify retention balances by project
  - Record any retention releases
  - Adjust for retention billing changes

- [ ] **Pending Change Order Review**
  - Update pending CO cost tracking
  - Reclassify approved COs
  - Write off rejected CO costs if applicable

### Intercompany Entries (if applicable)

- [ ] **Record Intercompany Transactions**
  - Equipment rentals between entities
  - Management fees
  - Allocated expenses
  - Verify intercompany balances zero

---

## Report Preparation (Day 9-10)

### Financial Statements

- [ ] **Run Preliminary Reports**
  - Balance Sheet
  - Profit & Loss
  - Cash Flow Statement (if prepared)

- [ ] **Review for Accuracy**
  ```
  Balance Sheet Review:
  - [ ] Total Assets = Total Liab + Equity
  - [ ] Bank accounts match reconciliations
  - [ ] AR balance reasonable
  - [ ] AP balance reasonable
  - [ ] Retention balances reconcile to detail
  - [ ] CIE/BIE matches WIP schedule
  - [ ] No unusual balances

  P&L Review:
  - [ ] Revenue matches billings (adjusted for WIP)
  - [ ] COGS reasonable vs. revenue
  - [ ] Gross margin within expected range
  - [ ] Overhead expenses reasonable
  - [ ] No unusual spikes or drops
  ```

### Job Cost Reports

- [ ] **Project Profitability Report**
  - Run for all active projects
  - Review margins by job
  - Identify problem projects
  - Note action items

- [ ] **Budget vs. Actual**
  - Run for active projects
  - Identify variances
  - Document explanations
  - Update budgets if needed

### Management Reports

- [ ] **Prepare Key Metrics Dashboard**
  ```
  KEY PERFORMANCE INDICATORS

  Revenue Metrics:
  - Backlog (contracts - billings):        $__________
  - Billings This Month:                   $__________
  - Billings YTD:                          $__________

  Profitability Metrics:
  - Gross Profit %:                        _________%
  - Net Profit %:                          _________%
  - Avg Job Margin:                        _________%

  Cash Metrics:
  - Cash on Hand:                          $__________
  - AR > 60 Days:                          $__________
  - AP Due < 30 Days:                      $__________

  WIP Summary:
  - Total Under-Billing:                   $__________
  - Total Over-Billing:                    $__________
  - Net Position:                          $__________
  ```

---

## Final Review & Close (Day 10)

### Review Meeting

**Meeting Participants:** Owner/CFO, Controller, Key Staff

- [ ] **Review Financial Statements**
  - Walk through Balance Sheet
  - Review P&L performance
  - Discuss significant variances

- [ ] **Review Job Performance**
  - Discuss problem projects
  - Review WIP analysis
  - Make decisions on action items

- [ ] **Approve Close**
  - Sign off on financials
  - Authorize period close
  - Note any known issues for next month

### Close the Period

- [ ] **Lock the Period (if desired)**
  - In QBO: Settings → Advanced → Close the books
  - Set closing date to last day of month
  - Require password to modify prior periods

- [ ] **Archive Reports**
  - Save PDF copies of key reports
  - Store in organized folder structure
  - Back up any Excel work papers

---

## Post-Close Activities (Day 11+)

### Report Distribution

- [ ] **Distribute Financial Package**
  - Send to owner/shareholders
  - File in company records
  - Provide to CPA if required
  - Submit to bonding company if due

### Action Items Follow-Up

- [ ] **Document Action Items**
  ```
  ACTION ITEMS - [MONTH]

  Project Issues:
  1. _________________________________
  2. _________________________________

  Collection Actions:
  1. _________________________________
  2. _________________________________

  System/Process Improvements:
  1. _________________________________
  2. _________________________________
  ```

- [ ] **Calendar Next Month Close**
  - Schedule WIP meeting
  - Set reminder for billing deadlines
  - Note any special items

## Troubleshooting Common Close Issues

### Out of Balance Situations

| Issue | Likely Cause | Resolution |
|-------|--------------|------------|
| BS doesn't balance | Missing entry, one-sided JE | Review journal entries |
| Bank rec won't balance | Timing, missing transactions | Review outstanding items |
| AR doesn't match detail | Payment application errors | Review customer balances |
| AP doesn't match detail | Unapplied payments | Review vendor balances |

### WIP Issues

| Issue | Likely Cause | Resolution |
|-------|--------------|------------|
| % complete seems wrong | Bad cost estimate | Meet with PM |
| Large over-billing | Front-loaded billing | May be OK, document |
| Large under-billing | Behind on billing | Bill immediately |
